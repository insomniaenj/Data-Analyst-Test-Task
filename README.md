# Data Analyst Test Task: Lead Sniper

## Цель проекта
Автоматизированный сбор базы российских компаний с отделом поддержки 10+ человек.

## Как запустить
1. Клонировать репозиторий: `git clone https://github.com/insomniaenj/Data-Analyst-Test-Task`
2. Установить зависимости: `pip install -r requirements.txt`
3. Запустить проект: `python main.py`

## Методология и подходы
- **Источники**: Использование HH.ru API (сбор вакансий в категориях Support/Helpdesk).
- **Логика 10+**:
    - **Уровень A**: Поиск прямых упоминаний численности через Regex в описаниях вакансий.
    - **Уровень B**: Эвристическая оценка на основе графиков 24/7 и сменности (считаем, что покрытие 24/7 требует штата > 10 чел).
- **Сбор ИНН**: Программное извлечение из юридических блоков HH API и мета-данных сайтов.

## Ограничения и допущения
- Часть сайтов блокирует автоматические запросы (использованы кастомные Header и отключение прокси).
- ИНН может отсутствовать, если компания не указала его в открытых источниках.
